Scope: this is a condensed v8 merge slice: V8 kernel first, with V7/prompt intentions re-added as compact modules. It has localdata vault, identity, manual WebRTC first contact, negotiated control/binary channels, binary file pump with backpressure, route-vault recording, bounded eval, SignalMux hooks, SyncPlanner, and MediaManager. It is still not the full production Auto-Rendezvous/Sync release.
