Info

This is an automatic diff app.

Format:

<coloured diff 7>
common anchor/*🟢*/new code/*🟡old code🔴*/common anchor
</coloured diff 7>

Use unchanged surrounding text as a unique anchor. Multiple distant edits can use separate tags; nearby related edits can share one tag when their middle unchanged anchor is sufficiently distinctive.

Add-only: 
anchor/*🟢*/new code/*🟡🔴*/anchor
Remove-only: 
anchor/*🟢*//*🟡old code🔴*/anchor

Test is read-only. Surgically Inject performs the same exact test and commits its resulting target. Failed cards expose Try Fixing, which checks condensed-whitespace and then whitespace-removed anchors; Update Diff converts a unique suggestion back into an ordinary exact diff.
